Everyone tourist site
====
地元のおすすめスポットを共有できるWebアプリケーションです。

## Description
Eclipseで作ったWebアプリケーションです。
ログイン認証、新規投稿、投稿閲覧、簡単な投稿検索機能などが実装されています。
